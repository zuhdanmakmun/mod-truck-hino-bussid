# Mod Truck Hino 500 700 BUSSID

Kumpulan mod truck Hino 500 & 700 untuk game Bus Simulator Indonesia (BUSSID).
Dikelola oleh **ZM Studio Labs**.

---

## Tentang Repositori

Repositori ini menyimpan data katalog mod, file mod `.bussidmod`, dan thumbnail
yang digunakan oleh aplikasi Android **Mod Truck Hino 500 700 BUSSID**.

## Struktur Folder

```
mod-truck-hino-bussid/
├── mods.json           → data katalog semua mod
├── config.json         → konfigurasi URL dan versi app
├── thumbnails/         → foto thumbnail setiap mod (JPG)
└── releases/           → catatan rilis (file mod di tab Releases)
```

## Kategori Mod

| Kategori | Keterangan |
|---|---|
| Hino 500 | Truk Hino 500 berbagai muatan |
| Hino 700 | Truk Hino 700 muatan berat |
| Hino Lohan | Truk Hino Lohan |
| Hino Dump | Truk Hino dump muatan batu/sawit |

## Cara Tambah Mod Baru (tanpa update app)

1. Upload file `.bussidmod` ke tab **Releases** → versi baru
2. Upload thumbnail JPG ke folder `thumbnails/`
3. Tambah entry baru di `mods.json`
4. Commit — app Android otomatis tampilkan mod baru
